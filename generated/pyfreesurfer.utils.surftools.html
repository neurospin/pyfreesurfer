class pyfreesurfer.utils.surftools.TriSurface(vertices, triangles, labels=None, metadata=None, inflated_vertices=None)[source]¶

A class representing a triangulated 3D surface.

The surface contains ‘ntrian’ triangles, each having 3 vertices with 3 coordinates.
